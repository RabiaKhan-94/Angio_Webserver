Read Me

  For the convenience of users, particularly for the vast majority of experimental scientists, a publicly accessible web-server for Prediction model for Angiogenesis and Tumor angiogenesis process has been established. Moreover, to maximize users' convenience, given below a step-by-step guide on how to use it to get the desired results without the need to through the above mathematical details.

  Step 1. Open the web-server by clicking the link at http://biopred.org/Prediction model for Angiogenesis and Tumor angiogenesis process/ and you will see its top page of Prediction model for Angiogenesis and Tumor angiogenesis process. Click on the ReadMe button to see a brief introduction about the server.

  Step 2. You can either type or copy and paste the query sequences into the input box at the center of Prediction model for Angiogenesis and Tumor angiogenesis process Prediction Page. The input sequence should be in simple sequence format or in the FASTA format. A potential sequence in FASTA format consists of a single initial line beginning with the symbol ">" in the first column, followed by lines of sequence data in which nucleotides are represented using single-letter codes. Except for the mandatory symbol ">", all the other characters in the single initial line are optional and only used for the purpose of identification and description. The sequence ends if another line starting with the symbol ">" appears; this indicates the start of another sequence.

  Step 3. Click on the Submit button to see the predicted results. The predicted results will be shown on the screen.

  Step 4. You can also download the predicted results into a file by clicking the Download button on the aforementioned screen.
